## 如果你刚好路过这里

我希望这个页面以后会慢慢变成一块真的留言板。

现在它还是静态的，但我已经给它留好了位置。等后面接入评论系统之后，这里可以用来放问候、建议、推荐，或者只是留下一句“我来过”。

## 我想把它做成什么样

- 不吵闹，但欢迎认真交流
- 可以留下长一点的话，也容得下短短一句
- 像深夜回到桌前时，顺手翻开的那一页纸

如果你想要，我下一步可以把这里接成 `Giscus` 或 `Waline`。
