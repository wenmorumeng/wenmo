## 这个静态博客已经具备什么

- 左侧信息栏和卡片式文章列表
- 顶部横幅和柔和的蓝色主题
- 夜间模式切换
- 文章页自动目录
- 归档页、关于页和留言板占位页

## 你后面最常改的地方

1. 修改 `docs/_config.yml` 里的博客标题、作者信息和公告。
2. 替换 `docs/assets/images/avatar.svg` 成你自己的头像。
3. 替换 `docs/assets/images/banner-day.svg` 和 `docs/assets/images/banner-night.svg` 成你喜欢的背景。
4. 在 `docs/_posts` 里继续新增 Markdown 文章。

## 一篇文章长什么样

文章文件名格式是：

```text
YYYY-MM-DD-文章英文标识.md
```

文件头部示例：

```yaml
---
title: 我的第一篇文章
date: 2026-04-06 20:00:00 +0800
categories: [文章]
tags: [生活, GitHub Pages]
cover: /assets/images/your-cover.jpg
excerpt: 这一段会显示在首页摘要里。
---
```

正文直接用 Markdown 写就可以。
